# 🐍 Практичні завдання Python для себе

У цій папці знаходяться особисті практичні завдання та невеликі проєкти для вивчення Python.  
Тут зберігаються різні експерименти, тренування та спроби реалізації власних ідей.

## 📚 Що є у папці

- Практика базового Python
- Робота з циклами, умовами та функціями
- Завдання на логіку
- Робота з файлами
- Невеликі консольні програми

## 🎯 Мета

- Покращення навичок програмування
- Закріплення теорії на практиці
- Вивчення нових можливостей Python

## 🛠️ Примітка

Деякі файли можуть бути тестовими або незавершеними, оскільки папка використовується для навчання та практики.
