# Types of compilation
## Домашнє завдання завдання за 10.11.25
### Є три файли: frstfile.cpp , scndfile.cpp , thrdfile.cpp .
Компілюємо їх усіх командами: 
```
g++ -c frstfile.cpp
g++ -c scndfile.cpp
g++ -c thrdfile.cpp
```
### В результаті ми отримали три файли: frstfile.o , scndfile.o , thrdfile.o .
## Щоб отримати готову виконувану програму я виконав наступну команду :
```
g++ -c frstfile.o scndfile.o thrdfile.o -o finalprogram
```
